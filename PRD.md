# Markdown 笔记应用产品需求文档 (PRD)

# 简要描述
这个项目要实现一个轻量级、跨平台的 Markdown 笔记应用，面向开发者、写作者和知识管理用户，提供流畅的编辑体验和强大的文档管理能力。用户可以创建笔记、输入标题、组织文件夹、添加标签、搜索笔记等。界面采用左右分栏，左侧为编辑区域，右侧为预览区域。左侧需要显示当前编辑的文件路径、文件名、标题、目录结构等，支持搜索。数据保存在LocalStorage，支持删除笔记。代码块要有语法高亮，这样看起来更专业

## 1 产品定位
一款轻量级、跨平台的 Markdown 笔记应用，面向开发者、技术写作者和知识管理用户，提供流畅的编辑体验和强大的文档管理能力。

## 1.2 目标用户
- **主要用户**：开发者、技术写作者、学生
- **次要用户**：内容创作者、项目经理、知识工作者

## 1.3 核心价值
- 🚀 **高效编辑**：实时预览、快捷键支持、自动保存
- 📁 **智能管理**：文件夹组织、标签分类、全文搜索
- 🎨 **优雅体验**：简洁界面、主题切换、专注模式
- 💾 **安全可靠**：本地存储、版本历史、数据导出

# 2 功能需求
## 2.1 基本功能
- **Markdown 编辑**：支持标准 Markdown 语法，实时预览
- **文件夹管理**：创建、组织、删除文件夹
- **标签分类**：为笔记添加标签，方便分类管理
- **全文搜索**：快速定位和检索笔记内容

### 2.1.1 编辑功能
| 功能项 | 优先级 | 描述 |
|--------|--------|------|
| Markdown 语法支持 | P0 | 支持标准 Markdown 语法（标题、列表、代码块、表格等） |
| 实时预览 | P0 | 编辑时同步显示渲染效果 |
| 分屏模式 | P0 | 支持编辑/预览左右分屏显示 |
| 所见即所得模式 | P1 | 可选的 WYSIWYG 编辑模式 |
| 代码高亮 | P0 | 支持多种编程语言的语法高亮 |
| 数学公式 | P1 | 支持 LaTeX 数学公式渲染 |
| 流程图支持 | P2 | 支持 Mermaid 流程图、时序图 |
| 表格编辑 | P1 | 可视化表格插入与编辑 |
| 图片插入 | P0 | 支持本地图片上传、粘贴、拖拽 |
| 快捷键 | P0 | 常用操作的键盘快捷键（加粗、斜体、插入链接等） |
| 自动保存 | P0 | 编辑内容自动保存，防止数据丢失 |
| 撤销/重做 | P0 | 支持无限撤销和重做操作 |

### 2.1.2 导入导出
| 功能项 | 优先级 | 描述 |
|--------|--------|------|
| Markdown 导入 | P0 | 导入 .md 文件 |
| 批量导入 | P1 | 批量导入多个 Markdown 文件 |
| 导出为 Markdown | P0 | 导出为 .md 文件 |
| 导出为 HTML | P0 | 导出为带样式的 HTML 文件 |
| 导出为 PDF | P1 | 导出为 PDF 文档 |
| 导出为图片 | P2 | 导出为 PNG/JPG 图片 |

#### 2.1.3 界面与体验

| 功能项 | 优先级 | 描述 |
|--------|--------|------|

| 自定义主题 | P2 | 用户自定义编辑器配色 |
| 字体设置 | P1 | 调整编辑器字体大小、字体类型 |
